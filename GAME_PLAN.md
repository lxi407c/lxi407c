# 게임 개발 계획서

## 게임 개요

| 항목 | 내용 |
|------|------|
| 제목 | Dodge the Rocks (가제) |
| 장르 | 캐주얼 모바일 |
| 플랫폼 | 웹 브라우저 (HTML5) |
| 엔진 | Godot 4 |

| 목표 기간 | 1~2일 |

---

## 게임 컨셉

위에서 떨어지는 장애물을 피하며 최대한 오래 살아남는 생존형 캐주얼 게임.
터치/마우스로 조작 가능하여 모바일 웹에서도 즐길 수 있음.

---

## 핵심 기능 (Must Have)

- [ ] 플레이어 이동 (터치 / 마우스 / 키보드)
- [ ] 장애물 생성 및 낙하
